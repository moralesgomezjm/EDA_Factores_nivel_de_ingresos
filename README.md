# EDA-Project-Template

Plantilla base para proyectos de EDA (Exploratory Data Analysis - Análisis Exploratorio de Datos) del Bootcamp de Data Science Online de The Bridge.

## Instrucciones de esta plantilla

Recordad que solo es necesario un repositorio por equipo. Para crearlo usando esta plantilla, el/la responsable del equipo debe seguir estos pasos:

1. **Haz clic en el botón verde "Use this template"** en la parte superior derecha de este repositorio
2. Selecciona **"Create a new repository"**
3. Configura el nuevo repositorio:
   - Añade el nombre que hayáis elegido para el proyecto, recordad que debe seguir la estructura EDA_<Temática_del_proyecto>, p.ej.: "EDA_Vivienda_Madrid"
   - Añade una descripción (opcional)
   - Selecciona que la visibilidad sea **pública**
4. Haz clic en **"Create repository"**
5. **Añade como colaboradores al resto del equipo**: Ve a Settings → Collaborators → Add people, e invita a tus compañeros usando sus usuarios de GitHub
6. Clonad el nuevo repositorio en vuestros respectivos equipos mediante línea de comandos o usando GitHub Desktop
7. Ya podéis empezar vuestro proyecto, y recordad leer el enunciado con calma y detenimiento
